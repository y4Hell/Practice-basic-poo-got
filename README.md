## GOT super basic example program

Ejercicio basico de Programación Orientada a Objetos en Java.

## ¿Qué hace?
- Personajes de Game of Thrones (Rey, Reina, Caballero, Troll, Dragón).
- Cada uno pelea de forma diferente (polimorfismo).
- Armas que se intercambian en tiempo de ejecución: Espada, Hacha, Cuchillo, Arco.

## Cómo ejecutarlo
Ejecuta la clase `Got.java`.
